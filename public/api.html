Panda API

Overview

Welocome to the Panda Public API! Below you will find brief descriptions of the data we have available. We encourage you to build your own dashboards or personalized learning portal using Panda's available resources and continue learning.

If you have any resources you'd like added to Panda, please share them by email to pandashelpers@gmail.com or send us a tweet to @panda_guide_app. We appreciate your contributions!

Resources
GET /api/v1/resources
Fetch a complete list of resources (our term for links to PDFs, Videos, and other materials).
GET /api/v1/resources/:technology
Fetch a list of all resources related to a specific technology.
technology: string [html, css, javascript, jquery, api, node.js, npm, mysql, sequelize, express.js, mongodb, react, hadnlebars, github, heroku, security, debug, testing, tools]
GET /api/v1/resources/:technology/:type
Fetch a list of resources related to a specific technology by specific type of resource.
type: string [video, pdf, link]
Users
GET /api/v1/users
Fetch a list of all registered Panda users. Users data includes all publicly avalailable data via Github (email, Github link, user name, etc.), limited to four code repositories chosen by the specified user.
GET /api/v1/users/:id
Fetch a registered Panda user by their ID. Users data includes all publicly avalailable data via Github (email, Github link, user name, etc.), limited to four code repositories chosen by the specified user.
